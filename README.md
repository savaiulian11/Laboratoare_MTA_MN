# 📊 Laboratorul 6: Modele de Regresie - Seturi de Date

* **`housing.csv`** (California Housing)
  * **Descriere:** Prețul mediu al caselor din California în funcție de locație, vechime, număr de camere și venitul mediu al locuitorilor.
  * **Ideal pentru:** Analiză de corelație (Heatmap), Regresie Liniară Simplă (ex: *Venit vs. Preț*) și Regresie Liniară Multiplă.

* **`student_study_hours.csv`**
  * **Descriere:** Relația dintre numărul de ore de studiu alocate unei materii și nota finală obținută la examen.
  * **Ideal pentru:** **Regresia Liniară Simplă** (O singură variabilă independentă de intrare). Un set de date perfect pentru a vizualiza o dreaptă de regresie clasică.

* **`student_performance.csv`**
  * **Descriere:** Performanța studenților evaluată pe baza mai multor factori (ore de studiu, activități extracurriculare, etc.).
  * **Ideal pentru:** **Regresia Liniară Multiplă**. Vă permite să vedeți cum performanța este influențată de o combinație de variabile.

* **`insurance.csv`** (Costuri Medicale)
  * **Descriere:** Costurile facturate de asigurările medicale, influențate de vârstă, sex, indicele de masă corporală (BMI), numărul de copii și statutul de fumător/nefumător.
  * **Ideal pentru:** **Regresia Liniară Multiplă**. Un exemplu excelent din viața reală pentru a demonstra cum adăugarea unor parametri (ex. fumător) schimbă drastic predicția.

* **`dev_salaries.csv`** * **Descriere:** Evoluția salariului în funcție de nivelul de experiență/funcția ocupată.
  * **Ideal pentru:** **Regresia Polinomială**. Demonstrează clar cum o creștere salarială nu este întotdeauna liniară (salariile explodează adesea la nivelurile de management superior), necesitând o curbă polinomială pentru o predicție exactă.

* **`auto_mpg.csv`**
  * **Descriere:** Eficiența consumului de combustibil (Miles Per Gallon) al diverselor modele de mașini din anii '70-'80, bazat pe greutate, cai putere, capacitate cilindrică etc.
  * **Ideal pentru:** **"All-in-One"**. Poate fi folosit pentru toate cele 3 tipuri de regresie (ex: liniară folosind greutatea, polinomială folosind caii putere).
